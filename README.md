# 问天 · AI 智囊团

> 遂古之初，谁传道之？——屈原《天问》
>
> 向各领域最高智者发问。

问天是一个 AI 智囊团技能，将各领域顶级思考者的认知方式蒸馏为可调用的专家 Skill，支持按需咨询和从素材蒸馏新专家。

## 特性

- **多专家咨询**：单专家深度分析 + 多专家圆桌讨论（2-3人多视角）
- **智能路由**：三级匹配——明确指定 → 关键词 → 语义理解
- **认知植入**：基于仓颉 v2 架构，注入信念形成故事、真实决策反思、失败教训、内在矛盾等不可替代的认知上下文
- **自蒸馏**：内置 7 步蒸馏流程，从任意素材（书/访谈/文章）创建新专家
- **可扩展**：新增专家只需添加一个 SKILL.md + 注册表条目

## 快速开始

```
# 指定专家
用芒格的方式帮我分析要不要投资这个项目

# 自动路由

# 圆桌讨论
AI 创业产品的投资价值，几个专家一起来看看
```

```
# 网搜蒸馏
帮我蒸馏刘润

用这些资料蒸馏张小龙的思维 Skill
```

## 专家格式：仓颉 v2

问天采用仓颉 v2（认知植入）架构蒸馏专家，核心理念：

> 不给模型写推理流程（模型已经会推理），只注入模型不可能自己知道的认知上下文。

每个专家 SKILL.md 包含：
- **信念及其来源**：不只是"他信什么"，而是"他为什么信这个"
- **关键决策**：真实案例 + 事后反思
- **栽过的跟头**：本人承认的失败
- **内在矛盾**：保留不调和，矛盾是人格最有辨识度的特征
- **认知边界**：明确不懂什么
- **思维习惯**：第一人称描述，非推理步骤

## 新增专家

### 自动蒸馏

对问天说「蒸馏XX」即可触发内置的 7 步蒸馏流程。支持上传素材或网搜采集。

### 手动添加

1. 参照 `templates/expert-template.md` 编写 `experts/{id}/SKILL.md`
2. 在 `expert-registry.md` 追加注册条目
3. 将参考素材放入 `experts/{id}/references/research/`
